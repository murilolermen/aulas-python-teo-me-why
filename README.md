# 🐍 Aulas de Python — Téo Me Why

[![Python Version](https://img.shields.io/badge/Python-3.x-blue.svg)](https://www.python.org/)
[![Status](https://img.shields.io/badge/Status-Em%20Desenvolvimento-green.svg)]()

Repositório destinado ao armazenamento de anotações, exercícios práticos e mini-projetos desenvolvidos durante o curso de **Python** do canal [Téo Me Why](https://www.twitch.tv/teomewhy), ministrado pelo **Téo Calvo**.

O objetivo principal deste repositório é documentar a minha jornada de aprendizado na linguagem, servindo tanto como fixação de conceitos quanto como guia de consulta rápida para o futuro.
